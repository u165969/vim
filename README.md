# vim 
## What Is This?
This is a set of my vim config files.
## How to Use?
First, run those commands in your shell.
```
$ git clone https://github.com/u165969/vim.git
$ cd vim
$ chmod 655 init.sh
$ ./init.sh
$ vim ~/.vimrc
```

Then, run 'PlugInstall' in your vim editor.
```
:PlugInstall
```

Finally, run those commands in your shell.
```
$ chmod 655 config.sh
$ ./config.sh
```
